# Bloomberg Query Language (BQL) API (bloomberg-query-language-bql)
Bloomberg Query Language (BQL) is a structured query language used to access Bloomberg's extensive financial data, analytics, and research. It provides a flexible and powerful interface for retrieving market data, fundamentals, analytics, and custom calculations across various asset classes.
